A standalone **FastAPI dashboard** for monitoring Coda document exposure, detecting sensitive content, and triggering remediation workflows through the Coda REST API.

**Tech:** Python, FastAPI, REST APIs, Security Monitoring

---

### [Scaleway-Audit-Sentinel](https://github.com/PranavBajpai07/Scaleway-Audit-Sentinel)

A security monitoring dashboard for **Scaleway Audit Trail** and authentication events, featuring rule-based anomaly detection, alert management, remediation actions, local audit logging, demo mode, automated tests, and Docker support.

**Tech:** JavaScript, Docker, Audit Monitoring, Security Automation

### [Order-API](https://github.com/PranavBajpai07/Order-API)

An **ASP.NET Core Web API** for browsing logistics order data loaded from CSV files, providing structured order and line-item responses along with data-quality diagnostics.

**Tech:** ASP.NET Core, C#, REST API, CSV Processing, Logistics Data

### [TraWell](https://github.com/PranavBajpai07/TraWell)

**Tech:** Angular, ASP.NET Core, TypeScript, C#

### [PharmacyApp](https://github.com/PranavBajpai07/PharmacyApp)
